Github - Como usar

'Cabeça de ovo'.. tu já esqueceu como se faz.

Bom vamos lá.. abra o seu Github!

Primeiro entre no Github, e no canto superior, lado direito, clique no sinal de '+' e depois em 'Novo repositório'

Após abrir a janela, preencha o campo 'Nome respositório' escolha o nome do seu repositório

No campo 'Descrição', faça uma breve descrição do seu repositório

Em seguida escolha como 'público' ou 'privado'

Por fim clique no botão 'criar'

Abra seu terminal - ctrl+shift+t

Vá para a página em que se encontra os arquivos que você quer enviar para o github

quando estiver dentro da pastar inicie o git - git + init

agora abra o seu vs-code - code

dentro do seu vsc procure pelo botão raiz(source control)

Depois procure nele por (...) 3 pontinhos, e clique nele

Vá até a última opção ou procure por algo semelhante a 'show git output' e clique nele

Após abrir uma janela procure por 'Terminal' e clique

Vá em seu github abra o seu diretório criado e procure por 'code ou codigo' e clique nele

Vai aparecer algumas opções, selecione 'https' e copie o código mostrado

Volte para o vs-code

Agora digite o comando do terminal 'git remote add origin' e gigite o link copiado

Pronto!

Vamos enviar o arquivo que você criou

Crie um arquivo qualquer

Olhe para a 'raiz' no lado esquerdo, vai aparecer um 'bolinha azul' com o número '1'

Clique na 'raiz' e procure pela 'seta de correto' e clique nela

A seta é o nosso 'commit' clique nela e vai aparecer a área para você inserir a mensagem. preencha e dê enter!
